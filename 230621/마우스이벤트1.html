마우스 이벤트

jQuery 사이트에서 마우스 이벤트 확인

  1. click
  2. dblclick
  3. mouseover
  4. mouseenter
  5. mouseout
  6. mouseleave
  7. hover: 마우스를 객체 위에 올렸을 때와 벗어났을 때. jQuery 전용

이벤트 적용시 

        $().on('이벤트명', function(){});
여러 이벤트 적용시 
        $().on({
          'mouseenter': function (){},
          'mouseleave': function (){},
          '이벤트명': function (){}
        })
